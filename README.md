### 相关依赖
使用 [imagemin](https://github.com/imagemin/imagemin) 实现图片压缩
使用 [images](https://github.com/zhangyuanwei/node-images) 实现图片修改大小
使用 [sharp](https://sharp.pixelplumbing.com) 实现图片格式转换，修改大小
